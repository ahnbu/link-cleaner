# [Design] 특수 문자 정제 로직 추가 (Special Character Cleansing)

## 1. 개요 (Overview)
`localCleansingService.ts`에 특정 특수 문자 및 이스케이프 패턴을 정제하는 로직을 설계함.

## 2. 변경 대상 (Target Files)
- `services/localCleansingService.ts`

## 3. 상세 설계 (Detailed Design)

### 3.1. 정제 규칙 정의
| 원본 패턴 (Regex) | 변환 결과 | 설명 |
| :--- | :--- | :--- |
| `\.` | `.` | 역슬래시로 이스케이프된 마침표를 일반 마침표로 복원 |
| `\(-{3,})` | `$1` | 역슬래시로 시작하는 3개 이상의 대시(구분선)에서 역슬래시 제거 |

### 3.2. 구현 위치
`localCleansingService.ts`의 `cleanse` 함수 내부에서 다른 정규표현식 처리 전후에 배치함. 
순서상 이스케이프 해제는 공백 정리 전이나 후에 수행해도 무방하나, 텍스트의 본래 의미를 복원하는 과정이므로 초기에 수행하는 것을 권장.

## 4. 검증 계획 (Verification Plan)
- **테스트 케이스 1**: `입력: "문장 끝\. \---- 구분선"` -> `결과: "문장 끝. ---- 구분선"`
- **테스트 케이스 2**: `입력: "구분선\--- 세개"` -> `결과: "구분선--- 세개"`
- **테스트 케이스 3**: `입력: "일반 마침표."` -> `결과: "일반 마침표." (변화 없음)`
