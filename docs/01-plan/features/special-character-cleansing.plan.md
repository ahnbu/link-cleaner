# [Plan] 특수 문자 정제 로직 추가 (Special Character Cleansing)

## 1. 개요 (Overview)
텍스트 내에 존재하는 불필요한 이스케이프 문자(`\.`)나 과도한 구분선(`\----`) 등을 정제하여 가독성을 높이고 데이터를 깔끔하게 유지하기 위한 로직을 추가함.

## 2. 목표 (Goals)
- **특수 패턴 정제**: `\.`를 `.`으로, `\----`를 `----`로 변환하는 정규표현식 로직 구현.
- **로컬 서비스 확장**: `localCleansingService.ts`에 해당 변환 규칙을 통합.
- **가독성 향상**: 불필요한 역슬래시 제거를 통해 사용자에게 최종적으로 깨끗한 텍스트 제공.

## 3. 대상 패턴 (Target Patterns)
- **이스케이프된 마침표**: `\.` -> `.`
- **이스케이프된 구분선**: `\----` (또는 `\---`) -> `----` (또는 `---`)

## 4. 상세 작업 내용 (Tasks)
- [ ] **`localCleansingService.ts` 수정**:
    - `\.` 정제 규칙 추가: `text.replace(/\./g, '.')`
    - `\----` 정제 규칙 추가: `text.replace(/\(-+)/g, '$1')`
- [ ] **유닛 테스트/검증**:
    - 해당 패턴이 포함된 샘플 텍스트로 클렌징 결과 확인.

## 5. 기대 효과 (Expected Benefits)
- 특정 AI 모델이나 마크다운 파서에서 발생하는 불필요한 이스케이프 문자 자동 제거.
- 텍스트 품질의 일관성 유지.
