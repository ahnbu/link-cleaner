결과 화면의 가독성을 높이고 사용자 액션을 유도하기 위해 요소의 배치 순서를 전면 재조정했습니다.

## 2. 주요 변경 사항
- **중요도 기반 재배치**: 액션 버튼(상단) -> 변환 결과(중단) -> 원본 비교(하단) 순으로 변경.
- **상단 고정 액션 바**: 스크롤을 내려도 복사 및 다운로드 버튼이 상단에 유지되어 긴 문서 작업 시 편리함 제공.
- **시각적 계층 구조**: `z-20`과 `backdrop-blur`를 활용하여 상단 고정 바의 입체감 확보.
